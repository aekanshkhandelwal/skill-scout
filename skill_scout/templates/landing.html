How it works

Collect → Normalize → Deduplicate → Score → Serve. Refresh whenever you want the latest trends.

Step 01

Collect

MCP Registry, GitHub SKILL.md ecosystem, NPM search, and local directory listings.

Step 02

Score

Calculates popularity (stars/forks) and recency to generate a real-time impact score.

Step 03

Use

Search, filter by category, or ask the AI for a curated best-of list for your project.

Detailed Workflow

What you actually do

1

Click Refresh to build/update the index.

2

Search and filter by type (skill, mcp_server, plugin, connector, api, hook).

3

Use Ask for short recommended lists.

4

Open results and use Install hints.

On serverless (Vercel), storage may reset on cold starts — run Refresh again to repopulate results.

LLM providers (optional)

“Ask” works without a model (local ranking), but you’ll get better results when you connect a provider.

NVIDIA

OpenAI-compatible gateway. Set OPENAI_BASE_URL + OPENAI_MODEL.

OpenAI

Use default API. Set OPENAI_API_KEY.

Ollama

Local models. Set OLLAMA_HOST + OLLAMA_MODEL.

Deploy

Works locally and on serverless hosts.

Vercel

On cold starts, call /api/refresh (or use the in-app Refresh button).

Render / Fly

Easier persistence. Keep SQLite on disk or switch to Postgres.

FAQ

Why do I sometimes see 0 results?

On serverless, storage resets. Click Refresh to rebuild the index.

Does it install things automatically?

No—Skill Scout shows install hints and links. You stay in control.

Is the Antigravity directory verified?

It’s treated as discovery only. Validate before installing.

How do I keep API keys safe?

Use environment variables (Vercel settings or local .env). Never commit secrets.
